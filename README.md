jorgen_say
==========

Jorgen's Best Norwegian Dad Jokes for foturne

INSTALL
=======
1. copy jorgen.dat into your fortune directory
2. copy bike.cow into your cowsay cows directory

EXECUTE
=======
$ fortune jorgen | cowsay -f bike
